# Interactive Quiz Application With Timer

An Interactive Quiz Application developed using Frontend Development technologies like HTML, CSS, and JavaScript.  

This project provides users with a dynamic quiz experience along with a countdown timer for each question.

---

# Features

- Interactive Quiz Interface
- Multiple Choice Questions
- Real-time Score Tracking
- Question Navigation
- Countdown Timer
- Responsive Design
- Instant Correct/Wrong Feedback
- Restart Quiz Option

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
